# 0.传送门
https://music-player-three-delta.vercel.app/

# 1. 最原始版本
位于 `musicPlayer-js` 目录下

使用原汁原味的 `html + css + js`


# 2. React优化版本
位于 `musicPlayer-react` 目录下

使用 `React (Next) + Less`， 同时将原先在前端写死的歌曲列表优化为通过接口获取，接口方法写在 `src/pages/api/queryMusic.ts`，歌曲列表作为资源存储在 `public/db` 中


## 启动方式
```cmd
npm i
npm run dev
```
